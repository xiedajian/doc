# 长度单位
```
	px
	em
	rem
	vh
	vw
```


### px
px是固定的像素,
用户的浏览器默认渲染的文字大小是“16px”


### em
相对尺寸
em相对于父元素字体的大小，rem相对于根元素字体的大小


### rem
相对尺寸
em相对于父元素字体的大小，rem相对于根元素字体的大小
根元素是谁？<html>元素
```
	html {font-size:20px}
	p {font-size:0.8rem}

```


### vh

viewpoint height，视窗高度，1vh=视窗高度的1%


### vw

viewpoint width，视窗宽度，1vw=视窗宽度的1%

100vw就是布局视口的宽度,
