# 聚合marker

MarkerClusterer标记聚合器用来解决加载大量点要素到地图上产生覆盖现象的问题，并提高性能。 主入口类是MarkerClusterer。

当marker数量特变多，十分密集的时候，会产生性能问题和堆积问题，

这时候做个聚合是十分有用的方式。
